### 📢진행 방식
1. 매주 월, 수, 금 2강씩 강의를 들어온다.
2. 강의에 수록된 문제를 스터디 시간동안 참고없이 다시 풀어본다.

### ✨커밋 규칙
- 자기 이니셜 폴더 안의 코드만 생성, 수정한다.
- master 브랜치 하나로 커밋한다.
- 커밋 메시지는 ```[이름] 내용``` 포맷으로 작성한다.
